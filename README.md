## 環境構築
1.java1.8をインストール
```sudo yum install java-1.8.0-openjdk.x86_64```


2.java1.8にアップデート
```sudo alternatives --config java```

⇒java1.8を選択


3.Gradleのインストール(Win10向け)

http://qiita.com/quwahara/items/9c4e5fbda421cfcb09ad


4.sourcetreeのインストール

http://tracpath.com/bootcamp/learning_git_sourcetree.html
